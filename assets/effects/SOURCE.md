# Desktop Effects Assets

This folder is reserved for local desktop-effect assets.

Current implementation:
- Sakura petals are drawn from small pixel matrices in `js/desktop-effects.js`.
- Stars, shooting stars, and Windows-style bubbles are generated with Canvas 2D.
- No external bitmap assets are currently required.

Optional CC0 sources considered for future enhancement:
- Kenney Particle Pack: https://www.kenney.nl/assets/particle-pack
- OpenGameArt Particle Pack mirror: https://opengameart.org/content/particle-pack-80-sprites

Notes:
- Do not use ripped assets from commercial games such as Deltarune. The star sprites here are original pixel matrices drawn in code, inspired only by general retro pixel-art conventions.

If external assets are added later, download them into this folder and record the file names, source URLs, license, and download date here.
